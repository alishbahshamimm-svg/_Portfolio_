# Alishbah.dev — Personal Portfolio

A premium, editorial personal portfolio website for **Alishbah Shamim**, Junior Full-Stack Developer.

Built with plain **HTML5**, **CSS3**, **Bootstrap 5**, and **Vanilla JavaScript** — no build tools, no frameworks, no dependencies to install.

## Run locally

Just open `index.html` in any modern browser:

```
open index.html            # macOS
xdg-open index.html        # Linux
start index.html           # Windows
```

Or serve it with a simple local server:

```
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Notes

- Bootstrap 5 and Google Fonts (Inter + Fraunces) are loaded from CDN in `index.html`.
- Replace `assets/images/portrait.jpg` with your own portrait if desired.
- Drop your CV as `resume/resume.pdf` — the "Resume" nav button already links to it.
- Project "Live Demo" / "Code" buttons use `#` placeholders — update them with your real URLs.

## Design

- Editorial, magazine-inspired layout — asymmetric hero, generous whitespace, serif display type (Fraunces) paired with Inter for body.
- Warm palette on a near-white background, restrained use of the accent (#F4B43C).
- Subtle motion only: scroll reveal, hover lift, animated skill bars, typing effect.
